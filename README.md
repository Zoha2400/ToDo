This is a react app.
Draggable ToDo List.

Install the `zip` file and extract it. Then, open your cmd/console and go to the ToDo List folder. 
In the folder type
 `npm i` or `npm install` for downloading and adding all npm packages.

For starting open console and type:
 `npm start`

Done!
